# AnnounceHandle

`type`  
_cref_: `T:Microsoft.UI.Reactor.Hooks.AnnounceHandle`

> **Learn more:** [Hooks](../../hooks.md), [Effects](../../effects.md)

## Summary

Handle returned by [UseAnnounce](UseAnnounce.md) ([guide](../../hooks.md)).
Provides an imperative [Announce](Announce.md) ([guide](../../hooks.md)) method for screen reader
live-region announcements plus a zero-size [Region](Region.md) ([guide](../../hooks.md)) element
that must be included somewhere in the component's rendered tree.
